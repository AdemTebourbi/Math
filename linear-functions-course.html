Cours Résumé : Les Fonctions Linéaires

Niveau : 1ère année secondaire (lycée)

1. C’est quoi une fonction ?

  • Une fonction, c’est une règle qui prend un nombre x et donne un seul résultat.
  • On la montre avec : une formule (ex. f(x) = 2x), un tableau, ou un graphique.

2. Une fonction linéaire

  • C’est une fonction simple : f(x) = ax, où a est un nombre fixe (le coefficient).
  • Exemple : f(x) = -3x → f(-2) = -3 × (-2) = 6.
  • Image de -2 = 6.
  • Antécédent de 6 = -2.

3. Proportionnalité

  • Un tableau de f(x) = ax est proportionnel : chaque sortie = a × entrée.
  • Exemple : f(x) = 2x → x = -1, 0, 2 donne f(x) = -2, 0, 4.

4. Trouver le coefficient a

  • Si tu connais x et f(x), alors a = f(x) / x.
  • Exemple : f(√3) = 2√6 → a = 2√6 / √3 = 2√2.
  • Donc f(x) = 2√2 x.

5. Le graphique

  • Une fonction linéaire est une droite qui passe par (0,0).
  • a (coefficient directeur) montre si elle monte (a > 0) ou descend (a < 0).
  • Exemple : f(x) = -2x → Points : (0,0) et (-1, 2).
  • f(-2) = 4 (vu sur le graphe).
  • Point important : Si un point A(xA, yA) vérifie yA = f(xA), alors A appartient à la représentation graphique de f.
  • Astuce : Trace avec 2 points éloignés pour être précis.

En bref

  • f(x) = ax : a multiplie x.
  • Tableaux proportionnels.
  • a = f(x) / x.
  • Graphique = droite depuis (0,0), et un point A est sur la droite si yA = f(xA).
